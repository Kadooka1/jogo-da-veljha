🎮 Jogo da Velha – Atividade de Desenvolvimento Web
📌 Descrição
Este projeto é uma atividade prática de HTML e CSS com o objetivo de criar uma versão estilizada do clássico Jogo da Velha (Tic-Tac-Toe).
A aplicação apresenta a estrutura do tabuleiro e os símbolos X e O, posicionados em algumas casas, demonstrando a utilização de elementos de marcação e estilização.

🚀 Tecnologias Utilizadas
HTML5 → Estruturação da página e elementos do jogo.

CSS3 → Estilização do tabuleiro, das marcações e do layout.

🎯 Objetivos da Atividade
Praticar a construção de layouts em HTML;

Utilizar flexbox para organizar os elementos do tabuleiro;

Aplicar estilizações personalizadas com CSS;

Representar graficamente um Jogo da Velha.

📂 Estrutura do Projeto
📁 projeto-jogo-da-velha
 ├── index.html   # Página principal do jogo
 ├── style.css    # Arquivo de estilos
🖼️ Demonstração
O jogo apresenta um tabuleiro 3x3 com divisórias azuis e uma linha vermelha de destaque, além de símbolos X e O estilizados em tamanho grande para fácil visualização.
